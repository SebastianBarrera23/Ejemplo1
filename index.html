Pasos de manejo de GIT/HUB por consola

  • Abrimos la consola CMD para ejecutar los comandos de GIT/HUB
  • Inicializamos nuestro repositorio con e comando GIT INIT
  • Verificamos que versión tiene nuestro GIT con el comando GIT --VERSION
  • Con el comando GIT STATUS verificamos en que estado se ecnuentra nuestro proyecto, como no hemos hecho nada va a salir un mensaje que no hemos adicionado ningún elemento.
  • Con comando GIT COMMIT ADD -M, le adicionaremos una confimación como la queramos llamar.
  • Luego abrimos en el navegador el GIT nos logueamos y creamos un nuevo repositorio con el mismo nombre del proyecto
  • Luego nos dan dos lineas de codigo, cogemos la primera que es para hacer la conexión remota
  • Con el comando GIT REMOTE verificamos si ya esta la conexión para hacer el push
  • Ahora cogemos la segunda linea despues de haber creado el repsitorio y le damos enter
  • Despues de eso con el comando GIT PUSH
  • Ahora verificamos con el comando GET STATUS y por ultimo vamos al GIT y verificamos si se nos añadio
